# Light detector on Arduino



Project that shows direction of light source on diodes on Arduino.



## Photo

![Project setup](photo.jpeg)



## Required elements

* Arduino Uno connected do PC
* breadboard
* 5x LED (same color)
* 5x resistor 300 Ohm
* 2x resistor 1 kOhm
* 2x photoresistor
* cables

Code file: light-detector.ino
