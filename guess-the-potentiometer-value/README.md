\# Guess the potentiometer value game on Arduino



Simple game, with diodes to show actuall state of game.



\## Photo

Picture of this project: photo.jpeg



\## Required elements

* Arduino Uno connected do PC
* 3x LED (red, yellow, green)
* 3x resistor 300 Ohm
* tact switch
* potentiometer
* breadboard
* cables
