# Guess the potentiometer value game on Arduino



Simple game with diodes to show the actual state of the game.



## Photo

![Project setup](photo.jpeg)



## Required elements

* Arduino Uno connected to PC
* 3x LED (red, yellow, green)
* 3x resistor 300 Ohm
* tact switch
* potentiometer
* breadboard
* cables
